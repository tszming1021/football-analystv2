## 五、未纳入计分的文件原因
- 赛前报告/推荐总表：可抽取预测，但缺少实际比分。
- GPT联网复核报告：多为一致性复核，不是最终赛果复盘。
- 本地 API fixtures 缓存：多数 World Cup fixtures 的 `goals.home/away` 为空，状态仍为 `Not Started`。
- 图片、docx、zip：未发现可直接结构化结算的“预测 vs 实际比分”表。
